В рамках выполнения домашнего задания необходимо создать модуль со следующим функционалом:
1. При установке модуля автоматически импортируется файл с множественным переводом слова «tasks».
a. Одна задача
b. Две задачи
c. …
d. Пять задач
2. По произвольному адресу выведен переведенный текст «__СЛУЧЧИСЛО__ задач(а/и)», где конечная формулировка зависит от значения случайного целого числа от 0 до 100.
3. По произвольному адресу вывести ссылку, при клике по которой отображается сообщение (любым образом) с датой регистрации текущего пользователя в формате “05 февраля 2013” и указанием времени (в днях) до Нового Года. При этом форматированная дата и количество дней передается на уровень Javascript при помощи drupal_add_js().
4. Создать страницу с textarea и кнопкой. При клике на textarea заменяется на введенный текст, при этом все специальные символы должны быть кодированы в строку.
