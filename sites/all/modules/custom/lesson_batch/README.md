Файлы, библиотеки
1. Создать страницу с таблицей и ссылкой, при клике по которой открывается модальное окно с формой.
a. На форме должен быть элемент для загрузки файлов.
2. В таблице выводится информация о всех загруженных файлах: порядковый номер, идентификатор пользователя (загрузившего файл), путь, статус  и ссылка для загрузки. 
3. Доступ на скачивание файлов должен быть ограничен тем, загружал ли данный пользователь когда-нибудь файлы.
4. Колонка «идентификатор пользователя» должна быть спрятана средствами CSS. 
5. При загрузке страницы средствами Javascript все строки с файлами, которые загрузил текущий пользователь, должны быть выделены каким-либо цветом.
6. Создать таблицу в базе данных для сохранения курсов валют.
7. Сделать batch-задачу для загрузки курсов валют с Нацбанка Беларуси.
8. Сделать View для отображения загруженных валют.

Рекомендации по выполнению
1. Страница регистрируется при помощи hook_menu.
2. Активирующая окно ссылка рендерится ctools_modal_text_button().
3. Файлы могут быть получены любым способом обращения к БД, например db_select(‘files_managed’).
4. При задании формы в качестве элемента загрузки файлов используется #type = ‘managed_file’.
5. CSS-файл подгружается при помощи drupal_add_css(drupal_get_path(‘module’, $module_name) . $path);
6. Javascript-файл подгружается при помощи drupal_add_js().
7. Номер пользователя достаётся средствами Javascript из конфигурационного массива Drupal.settings, а туда помещается посредством drupal_add_js(array(‘uid’ => $uid, ), ‘setting’).
8. Обработка таблицы начинается в функции Drupal.behaviors.some_behavior.
9. Во время импорта курсов валют выполняется лишь одна batch-операция.
10. Таблица интегрируется с Views посредством hook_views_data().
